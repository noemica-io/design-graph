01 · SPAWN

# spawn

fan out N isolated agents. each reads the brief independently — no peeking at siblings, so directions actually diverge.

$ design-graph spawn "results page" --n 8
02 · COMPARE

# compare

variants land on a zoomable canvas as they finish. drag to cluster. split-view two siblings at 1:1 scale. no pre-commit to a single direction.

$ design-graph canvas --live
03 · ITERATE

# iterate

pick any node, branch deeper. nudge copy, tighten type, swap the hypothesis. the graph keeps the ancestry so you can always rewind.

$ design-graph branch editorial-serif --n 4
